Первая страница HTML CSS JavaScript
HTML (стандартизированный язык гипертекстовой разметки документов):
  • Наполнение документа (страницы сайта);
  • Первичное расположение элементов;
  • Элементы (тэги): div, img, ul и тд.
CSS (формальный язык декодирования и описания внешнего вида документа):
  • Стиль каждого элемента;
  • Размеры элементов;
  • Позиционнирование элементов.
JS (мультипарадигменный язык программирования):
  • Динамичность страницы;
  • Анимация элементов;
  • Связь frontend'а (страницы) и backend'а (сервера).
